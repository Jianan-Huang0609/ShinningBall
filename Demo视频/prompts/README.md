# ShiningBall 极梦 Prompts 目录

本文件夹用于**产品图风格锚点**与**三场景 Demo 视频**的 prompt，按固定顺序使用可保证产品形象与画面风格一致。

---

## 文件与使用顺序

| 顺序 | 文件 | 用途 | 产出 |
|------|------|------|------|
| 1 | **01_产品图_风格锚点.md** | 生成产品主形象 | 产品风格锚点图（必保存） |
| 2 | **02_视频风格锚点_首帧.md** | 生成各场景首帧图（需带入产品锚点图） | 场景一 D1/D2、场景二 P1/P2、场景三 A1/A2 的锚点图 |
| 3 | **极梦_三场景_demo视频_prompts.md** | 生成三条 Demo 视频（需带入产品锚点 + 对应场景首帧锚点） | 三条视频成片 |

---

## 推荐操作流程

1. **产品图**：打开 `01_产品图_风格锚点.md`，用「必出：主锚点图」的 prompt 在极梦生成图，选一张最满意的保存为「产品风格锚点图」。
2. **视频风格锚点**：打开 `02_视频风格锚点_首帧.md`，在极梦中**上传产品锚点图**后，分别生成场景一的 D1、D2，场景二的 P1、P2，场景三的 A1、A2；各场景保存 1–2 张作为该场景的「视频风格锚点图」。
3. **生成视频**：打开 `极梦_三场景_demo视频_prompts.md`，对每条视频：**上传产品锚点图 + 该场景视频风格锚点图**，再按镜号（D1–D7 / P1–P7 / A1–A6）复制「极梦用」prompt 生成片段或整段，最后剪辑成片。

---

## 产品形态速查

- 矮圆柱、**织物/磨砂机身**、**顶环 RGB 灯带** + **小圆屏**（有屏幕感）、**摄像头**、内置音响
- **灵活转动**（Pan-Tilt 云台），**灯光样式多**、可打在场景/人/锚点位置，带科技感与屏幕感
- 参考 Apple HomePod / 小米 Sound；详见 PROJECT_DOC Part 1.5、产品图 prompt 见 `01_产品图_风格锚点.md`
