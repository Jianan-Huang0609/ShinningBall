# ShiningBall 视频风格锚点 · 各场景首帧

**用途**：为三条 Demo 视频分别生成「视频风格锚点图」（每场景取前 2 帧）。生成时**必须先带入 `01_产品图_风格锚点.md` 中产出的产品锚点图**（织物机身、顶环灯、小屏、摄像头、灵活转动），保证画面里的设备与产品图一致。保存后的首帧图在**步骤 3 生成视频时**与产品锚点图一起上传，用于锁定画面风格与产品形象。

**使用顺序**：
1. 已完成：用 `01_产品图_风格锚点.md` 生成并保存**产品风格锚点图**。
2. 本步骤：在极梦中**上传产品锚点图**作为参考，再分别生成下面各场景的**首帧 1 + 首帧 2**（即 D1/D2、P1/P2、A1/A2）。
3. 保存：场景一保存为「场景一_视频风格锚点_D1D2」、场景二「场景二_视频风格锚点_P1P2」、场景三「场景三_视频风格锚点_A1A2」。
4. 下一步：在 `极梦_三场景_demo视频_prompts.md` 中生成视频时，带入**产品锚点图 + 本场景对应的首帧锚点图**，再按镜号或整段生成。

```
no distorted hands, no extra fingers, no readable phone screen text, no brand logos,
```

```
Same device as product anchor (fabric cylinder, top ring, small screen, camera). Premium smart home, 16:9.
NEGATIVE.
```

### D2（Close）手机发指令 + 一体机亮起

**极梦用（生成时请带入产品锚点图）：**

```
Close-up of a smartphone in hand, screen shows a chat interface with one sent message bubble (blurred, no readable text), thumb just tapped send, in the background the smart speaker device top ring starts glowing with a soft breathing amber pulse, small circular screen shows a minimal mode icon, intimate moment of human commanding the device, shallow DOF.
```
